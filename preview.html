Claude Terminal Dashboard — UX Preview

Status indicators

Waiting — amber blink (needs user input)
Working — green pulse (Claude is active)
Done — cyan static (fades to idle after 30s)
Idle — no dot (terminal open, no activity)

Theme colors (left border)

Cyan — Life & finance
Green — Home & IoT
Blue — Knowledge & research
Magenta — Web & design
Yellow — Dev infrastructure
White — Root workspace

Interactions

Click tile → switches to that terminal
Active terminal gets highlighted background (see "api" tile)
Arrow keys / j/k → navigate tiles, Enter → switch
Sort order: waiting → working → done → idle, then by recency

Review questions

1. Are the tile proportions / padding right for your sidebar width?
2. Do the status dot animations read well? (pulse vs blink distinction)
3. Should "Idle" tiles show a status line at all, or just the name?
4. Is the left border thick enough to read the color at a glance?
5. Any projects missing from the theme color map?
